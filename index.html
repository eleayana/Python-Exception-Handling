a)Exception Handling

Exception means error detected at the time of program execution. However , while running a program , python generates an exception that should be handled to avoid your program to crash.

Examples:(Division by Zero,Addition of Two incompatible types, Accessing a file which doesn't exist).

Syntax:

try:
     #Code to execute
except:
      Code to execute in case of Errors
 else: 
     Code to execute in case of No Error
finally:
       code to execute in all cases

1st case: You are driving the road & the road is clear & you reach your destination without any trouble. This is called executing a program without exceptions.

2nd case: You are driving on the road and you see the accident on the road what you didn't expect to happen called exception.And your action of taking detour called Handling Exeptions.

Syntax:

try:
    while road_is_clear()
          drive()
except: Accident as e:
           take_detour()

# Class Accident(Exception) 
 class Accident(Exception):
    def __init__(self, msg):
         self.msg = msg
    def print_exception(self):
       print("Userdefined Exception:")
try: 
    raise Accident("Crash between 2 cars")
except Accident as e:
    e.print_exception()
    print(e)
